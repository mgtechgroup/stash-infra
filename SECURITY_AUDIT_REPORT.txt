## 2. CODE & DEPENDENCY VULNERABILITIES ⚠️

### NPM Vulnerabilities (Ophiuchi Desktop)

**Current Status**: 2 moderate severity

#### Vulnerability Details

**1. PostCSS (Moderate Severity)**
- **CVE**: GHSA-7fh5-64p2-3v2j, GHSA-qx2v-qp2m-jg93
- **Severity**: Moderate
- **Issue**: Line return parsing error, XSS via unescaped </style>
- **Fix**: Update PostCSS to ≥8.5.10
- **Status**: Can be fixed by upgrading Next.js to latest

**2. Next.js (High Severity)**
- **CVE**: GHSA-q4gf-8mx6-v5v3
- **Severity**: High
- **Issue**: Denial of Service with Server Components
- **Fix**: Update Next.js to ≥15.1.3
- **Status**: Requires version bump from 13.4.16 → 15.x (breaking change)

#### Remediation

**Option A: Conservative (Minimal Breaking Changes)**
```bash
cd repos/docker-srv/ophiuchi-desktop
npm install postcss@latest --save-exact
# This updates PostCSS only, keeps Next.js 13.x where possible
npm audit fix
```

**Option B: Complete (Recommended)**
```bash
cd repos/docker-srv/ophiuchi-desktop
npm install next@latest postcss@latest
npm install  # Install dependencies for new Next.js
npm test     # Verify functionality
```

#### Deployment Impact
- **Container builds**: Will pull latest npm dependencies
- **Breaking changes**: Next.js 15.x may require code updates (check `/app/pages`, API routes)
- **Timeline**: Can be deferred if no production Ophiuchi deployment
