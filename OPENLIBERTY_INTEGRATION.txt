## 🚀 DEPLOYMENT

### Start Stack with Open Liberty

```bash
cd stash/
docker compose -f docker-compose.unified.yml up -d
```

### Verify Liberty is Running

```bash
# Check container status
docker compose -f docker-compose.unified.yml ps liberty

# View logs
docker logs -f liberty

# Test health check
curl http://localhost:9080/health

# Via Nginx (HTTPS)
curl -k https://localhost/liberty/health
```

```yaml
    TZ: UTC
    LANG: en_US.UTF-8
    JAVA_OPTS: "-XX:+UseContainerSupport -XX:MaxRAMPercentage=75.0 -XX:+UseStringDeduplication"
    WLP_OUTPUT_DIR: /opt/ol/wlp/output
    DB_PASSWORD: uniscrape
```

### Server Configuration (server.xml)

Key sections:
- **HTTP Endpoint**: Port 9080, all interfaces
- **Features**: WebProfile-10.0, MicroProfile 5.0, OpenAPI 3.1, JWT SSO
- **Datasource**: PostgreSQL auto-configured
- **Security**: SSL/TLS (9443), JWT, BasicRegistry
- **Logging**: JSON format, auto-rotation
- **Health Checks**: MicroProfile Health 4.0

### Modify Configuration

Edit `stash/open-liberty/server.xml`:

```xml
<!-- Change port -->
<httpEndpoint port="8080" />

<!-- Add new datasource -->
<datasource id="MyDataSource" jndiName="jdbc/MyDS">
    <!-- Config -->
</datasource>

<!-- Add feature -->
<featureManager>
    <feature>graphQL-1.0</feature>
</featureManager>
```

Restart Liberty:
```bash
docker compose -f docker-compose.unified.yml restart liberty
```

## 💻 APPLICATION DEVELOPMENT

### Deploy WAR Files

**Option 1: Copy to dropins (Auto-deploy)**
```bash
docker cp myapp.war liberty:/opt/ol/wlp/usr/servers/defaultServer/dropins/
```

**Option 2: Update server.xml**
```xml
<webApplication location="myapp.war" contextRoot="/myapp" />
```

**Option 3: Build with Docker**
```dockerfile
FROM open-liberty:latest
COPY myapp.war /opt/ol/wlp/usr/servers/defaultServer/apps/
```

### REST API Example

```java
@Path("/api/users")
@Produces(MediaType.APPLICATION_JSON)
public class UserResource {
    @GET
    public List<User> getUsers() {
        // Query database
    }
    
    @POST
    public Response createUser(User user) {
        // Save to database
        return Response.status(201).entity(user).build();
    }
}
```

### Database Access

```java
@Resource(lookup = "jdbc/DefaultDS")
private DataSource ds;

public void saveUser(User user) throws SQLException {
    try (Connection conn = ds.getConnection();
         PreparedStatement pstmt = conn.prepareStatement(
            "INSERT INTO users (name, email) VALUES (?, ?)")) {
        pstmt.setString(1, user.getName());
        pstmt.setString(2, user.getEmail());
        pstmt.executeUpdate();
    }
}
```

## ⚠️ TROUBLESHOOTING

### Application Not Deploying

```bash
# Check deployment log
docker logs liberty | grep -i "deploy\|war"

# Verify WAR file location
docker exec liberty ls -la /opt/ol/wlp/usr/servers/defaultServer/dropins/

# Check file permissions
docker exec liberty chmod 644 /opt/ol/wlp/usr/servers/defaultServer/dropins/*.war
```

### Database Connection Issues

```bash
# Test database connectivity
docker exec liberty curl postgresql://postgres:5432

# Check datasource configuration
docker exec liberty cat /opt/ol/wlp/output/defaultServer/logs/messages.log | grep -i datasource

# Verify PostgreSQL is running
docker ps | grep postgres
```

### High Memory Usage

```bash
# Check JVM heap
docker exec liberty jps -lmv

# Reduce heap percentage (edit docker-compose.yml)
JAVA_OPTS: "-XX:MaxRAMPercentage=50.0"

# Restart
docker compose -f docker-compose.unified.yml restart liberty
```

### SSL Certificate Errors

```bash
# Regenerate certificates
docker exec liberty rm -f /opt/ol/wlp/output/defaultServer/resources/security/key.p12

# Restart server
docker compose -f docker-compose.unified.yml restart liberty
```

### Nginx Health Check Failing

```bash
# Verify endpoint responds
curl -v http://localhost:9080/health

# Check nginx error log
docker logs nginx-proxy | grep -i health

# Update health check URL in server.xml if needed
```

## 🔄 INTEGRATION WITH STACK SERVICES

### Call UniScrape from Liberty

```java
@Inject
@RestClient
UniScrapeClient scrapeClient;

public String scrapeUrl(String url) {
    return scrapeClient.scrape(url);
}
```

REST client interface:
```java
@RegisterRestClient(baseUri = "http://uniscrape:8085")
public interface UniScrapeClient {
    @POST
    @Path("/scrape")
    String scrape(@QueryParam("url") String url);
}
```

### Call Stash from Liberty

```java
@Inject
@RestClient
StashClient stashClient;

public List<Scene> getScenes() {
    return stashClient.listScenes();
}
```

### Share PostgreSQL Database

Liberty and UniScrape share the same PostgreSQL instance:
```yaml
postgres:
  environment:
    POSTGRES_DB: uniscrape
    POSTGRES_USER: uniscrape
    POSTGRES_PASSWORD: uniscrape
```

Liberty datasource:
```xml
<datasource id="DefaultDataSource" jndiName="jdbc/DefaultDS">
    <properties.postgresql
        serverName="postgres"
        databaseName="uniscrape"
        user="uniscrape"
        password="${DB_PASSWORD}"/>
</datasource>
```
